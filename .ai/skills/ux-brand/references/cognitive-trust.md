# Cognitive & Trust Frameworks

## Perceptual (inform token derivation)

| Framework | How /ux-brand Uses It |
|---|---|
| **Aesthetic-Usability Effect** (Kurosu & Kashimura) | Attractive interfaces are perceived as more usable. BUT "attractive" is audience-relative — an artisan's "attractive" ≠ a designer's. Calibrate aesthetic level to audience expectation, not design trends. |
| **Mere Exposure Effect** (Zajonc) | People prefer things they've seen before. → Use visual patterns the audience already encounters daily (familiarity anchors). Don't innovate visually for low-tech — innovate functionally. |
| **Processing Fluency** | Easy-to-process visuals feel more trustworthy. High contrast, familiar fonts, conventional layouts → higher trust. Novel layouts, thin fonts, low contrast → lower trust for unfamiliar audiences. |
| **Von Restorff Effect** | Distinctive items are better remembered. → Make ONE key CTA per screen distinctly different (size + color + weight). Don't make everything distinctive — defeats the purpose. |
| **Cognitive Load Theory** (Sweller) | Intrinsic (task complexity) + Extraneous (bad design) + Germane (learning). Brand decisions affect extraneous load: unfamiliar fonts, unexpected colors, ambiguous icons ALL add load. |
| **Dual Coding** (Paivio) | Visual + verbal encoding is stronger. → Icon+label > icon-only OR label-only. For low-tech: label is primary, icon is supplementary reinforcement. |

## Trust (inform palette and tone)

| Framework | How /ux-brand Uses It |
|---|---|
| **Stanford Web Credibility** (Fogg) | Professional appearance → credibility. For B2B micro-business: "professional" = clean, institutional, reminiscent of bank/government. NOT minimalist startup aesthetic. |
| **Trust Pyramid** | Competence (clean layout) → Reliability (consistent design) → Integrity (transparent labels/pricing). Brand tokens must serve all three levels. |
| **Cialdini's Persuasion** (visual) | Authority: institutional colors, serif fonts. Social proof: counter/testimonial styling. Scarcity: urgent accent. Consistency: uniform design system. Liking: warm palette. |
| **Norman's 3 Levels** | Visceral (first impression: color, shape, cleanliness). Behavioral (usability: does it work?). Reflective (identity: "my kind of tool"). Brand targets visceral + reflective. |

## Application Rules

1. **Low tech-comfort audience**: Processing fluency + mere exposure dominate. Use FAMILIAR visual language above all.
2. **High tech-comfort audience**: Aesthetic-usability + Von Restorff matter more. Can innovate visually.
3. **Trust-critical contexts** (money, legal, health): Stanford credibility + trust pyramid. Institutional palette, professional typography. Avoid playful signals.
4. **First-use contexts**: Norman's visceral level — 50ms judgment. Clean, professional, appropriate to audience expectation.
5. **Repeated-use contexts**: Norman's behavioral level takes over. Consistency and predictability > visual impact.
